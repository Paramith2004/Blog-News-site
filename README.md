## 📌 Project Overview

PulseNews is a fully responsive frontend news website covering four categories:
- 💻 Technology
- 🌍 World News
- 🌿 Lifestyle
- 🔬 Science

Built as **Project 1** of the DecodeLabs Full Stack Development internship program — focused on mastering responsive frontend fundamentals before moving to backend development.

## 🎯 Project Goals

- Build a responsive frontend interface for a simple web application
- Master HTML, CSS, and JavaScript fundamentals **without frameworks**
- Apply mobile-first design principles
- Follow 2025 UI/UX standards with warm, grounded aesthetics
- Ensure WCAG accessibility compliance

## ✨ Features

- ✅ Responsive layout — mobile, tablet & desktop
- ✅ Semantic HTML5 landmarks (`header`, `nav`, `main`, `article`, `footer`)
- ✅ Mobile-first CSS with media queries (768px, 1024px)
- ✅ CSS Grid for page layout, Flexbox for components
- ✅ `clamp()` for fluid typography
- ✅ Hamburger menu for mobile navigation
- ✅ Active nav highlight on scroll (IntersectionObserver)
- ✅ Newsletter email subscription with validation
- ✅ Scroll-triggered card fade-in animations
- ✅ Sticky header with scroll shadow
- ✅ WCAG accessibility (aria labels, semantic roles, live regions)

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|------------|--------|
| `< 768px` (Mobile) | Single column, hamburger menu |
| `≥ 768px` (Tablet) | 2-column grid, horizontal nav |
| `≥ 1024px` (Desktop) | 3-column grid, full layout |
